为 Windows 系统安装 Codex（保姆级教程）

这篇文章记录在 Windows 系统中安装 OpenAI Codex 的完整流程：先准备 Git 和 Node.js， 再通过 npm 安装 Codex，最后完成登录、版本检查和在 PyCharm 中打开使用。

1. Git 安装

1.1 查看自己的系统架构

按 Win + R，输入 msinfo32 可以直接查看系统信息。

系统信息如下：

1.2 Git 下载

可以直接下载 Windows 系统 x64 架构 Git 安装包，或者进入 Git 官网 选择适合自己系统的版本。我的电脑是 x64，所以直接单击下载。

下载成功后直接安装。

其它步骤保持默认，点击 Next 即可。

1.3 检查 Git 是否安装成功

按 Win + R，输入 cmd 打开终端。

输入：

git --version

2. 下载与安装 Node.js 节点

2.1 下载节点

可以直接下载 Windows 系统 Node.js 官方 LTS 版，或者进入 Node.js 官方下载页面。

2.2 安装节点

然后会进入终端，按任意键即可；如果出现弹窗，全部选择“是”。

最后进入自动安装，等待安装完成。

2.3 检查

输入以下指令检查是否安装成功：

node -v
npm -v

出现版本号即可。

3. Codex 安装

3.1 安装 Codex

将下列代码复制到终端进行安装：

npm install -g @openai/codex

3.2 检查版本

codex --version

3.3 登录 Codex

直接在终端输入 codex。第一次会进入登录界面。

输入 1。

进入如下页面：

系统一般会直接打开登录页面。如果没有打开，就把终端里的链接复制到浏览器， 然后登录自己的 GPT 账号即可。

登录成功后的界面如下：

输入 hello，可以看到 Codex 回复 Hey! What can I help with today?。

到此结束。

3.4 切换账号

如果想切换账号，请在终端直接输入：

codex login

4. 在 PyCharm 打开 Codex

首先使用 PyCharm 打开项目文件，然后再打开终端。
